# MURA Disease Detection

A deep learning-based web application for detecting musculoskeletal abnormalities using medical images.

## Features

- Upload X-ray images
- Predict abnormality (Positive / Negative)
- Built with Flask backend and HTML frontend
- Uses trained DenseNet model

## Tech Stack

- Python
- Flask
- TensorFlow / Keras
- OpenCV
- HTML, CSS, JavaScript
